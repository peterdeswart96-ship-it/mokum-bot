Stap 1 — SAS token aanmaken (Peter doet dit)
1
Ga naar Azure Portal
portal.azure.com → Storage accounts → mokumbotrg904a → Containers → kennisbronnen
2
Klik op Generate SAS
Bovenaan de container pagina — niet via de storage account zelf, maar direct op de container
3
Stel de rechten in
Permissions: Read, Write, List, Delete — Expiry: 12 juni 2027 → klik Generate SAS token and URL
4
Kopieer de Blob SAS URL
De onderste URL (begint met https://mokumbotrg904a.blob.core.windows.net/kennisbronnen?sv=...) — die geef je aan Mark
Stap 2 — Storage Explorer installeren (Mark doet dit)
1
Download Storage Explorer
azure.microsoft.com → Storage Explorer → Download Windows → installeren & openen
2
Klik op Connect (stopcontact-icoon linksboven)
Kies "Blob container" → klik Next → kies "Shared access signature URL (SAS)"
3
Plak de SAS URL van Peter
Plak de volledige URL in het veld "Blob container SAS URL" → Next → Connect
4
Bestanden beheren
Sleep bestanden naar binnen, verwijder of bekijk — werkt als Windows Verkenner. Direct actief na uploaden.
